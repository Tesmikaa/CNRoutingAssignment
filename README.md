CN Routin gAssignment

This repository contains the technical documentation and project files for the Computer Networks (CN 201.3) module assignment. The project explores the implementation of network infrastructure, security, and routing protocols using Cisco Packet Tracer.

Project Overview

This assignment focuses on the structured design and implementation of an enterprise-grade network. It covers the transition from basic device security to advanced dynamic routing configurations. 

Tasks Included

Task 1: Basic Router Configuration

Focuses on foundational security, including hostname assignment and console password protection. 

Task 2: Static Routing Implementation

Demonstrates manual control over network traffic between the Computing, Business, and Science network segments.

Task 3: Dynamic Routing Configuration

Covers the implementation of RIP (Routing Information Protocol) and EIGRP (Enhanced Interior Gateway Routing Protocol) to automate route discovery.  

Repository Contents

- Task1.pkt - Topology file for Task 1: Basic Router Configuration
- Task2.pkt - Topology file for Task 2: Static Routing Implementation
- Task3.pkt - Topology file for Task 3: Dynamic Routing (RIP & EIGRP)
- AssignmentReport.pdf - Comprehensive technical report detailing all configurations and verification results
